# Practica-final-arquitectura-del-software

Realización de la práctica final de arquitectura del Software.

En la parte práctica no se han implementado las siguientes tablas debido a que no son mencionadas en ningún endpoint ni en otra parte del enunciado. Simplemente se asigna una sala principal a la actividad y no salas secundarias.
  -Una Actividad puede usar una única Sala principal, pero también puede requerir otras Salas secundarias, compartidas con otras actividades si no hay solapamiento (N a N).
  -Cada Sala tiene un único responsable técnico (ResponsableSala) (1 a 1).
